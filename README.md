## 🧠 What Each Project Was Built to Answer

I picked each project to deliberately cover a different concept, not to pad a list:

- **Can I build a clean API from scratch in an unfamiliar language?** → ai-native-journey
- **Can I get structured, reliable output from an LLM for a real workflow?** → ai-pr-reviewer
- **Do I understand RAG well enough to build it without a framework doing it for me?** → ai-customer-support-bot
- **Can I build something that reasons autonomously, not just follows steps I wrote?** → ai-research-agent
- **Can I handle real file input, chunking, and multi-turn memory?** → ai-document-analyser
- **Can I think beyond one service — design something a real team would architect?** → streamlit-ui-integrated-ai-prj
- **Do I understand the difference between an agent and a team of agents?** → ai-analyst-crew
